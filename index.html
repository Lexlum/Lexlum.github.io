Impact Snapshot

车载主线项目的结果面板：业务效果、数据缩减与量产落地集中看，不在左侧重复展开过程。

    95% 业务测试效果指标
    43% 训练数据缩减
    量产 智慧聆听车载落地

    基于 A800 百卡集群，主导 7B-110B Dense 模型与 235B MoE 模型演进训练。

    • 引入 CP、SP、EP 并行策略和 Packing 策略，模型效率提升 18%。
    • MFU 达到 37%，小模型 Function Call 在真实场景准确率提升 23%。

    适配 MindSpeed-LLM、LLaMAFactory、DeepSpeed-Chat 等训练框架，构建容器环境并适配调度启动方式。

    • 支持 7B / 14B 模型训练，SFT 变长序列任务 MFU 接近 20%。
    • MindIE 部署 DeepSeek、Qwen3-235B-A22B 等 MoE 模型，接入大模型应用平台。
    • 首个交付项目达到千万级收益，支持私有化部署并可快速复制。

    实现 MCPhub 代理，支持多 MCP 服务集成、分组管理、自定义工具描述、单工具禁用和 Toollist 缓存。

    • 支持 npm、uvx、sse、streamhttp、dxt 等方式加载 MCP。
    • 基于 Prometheus + Grafana 构建实时监控与告警体系。
    • 车控 Agent 上线使用，融合物模型快出、长短期记忆、车辆状态和 MCP list 筛选。

    调研主流端侧芯片并测试 LLaMA.cpp、MCL-LLM，多进程大核绑定提升 3.3 倍吞吐，首 Token 延迟降低 67%。

    • 会议本端侧处理流程整体耗时提升 3.8 倍，1 万字会议从 35 分钟降至 8 分钟。
    • 车载端侧垂域分类、敏感词检测、指代改写和复杂车控可在 0.5s 内完成。
